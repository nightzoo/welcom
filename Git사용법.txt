1. 저장소 만드는 방법
> git init

2. 버전 만들기
 Working tree
 Staging Area
 Repository
> git status
: git의 현재상태
Untraced (추적하지 않는 파일) 이 존재하는 경우, 버전 만들기가 필요함
> git add 파일명
: 파일명을 Staging Area에 올림
> git status
Changes to be committed (커밋할 변경 사항) 으로 변경
> git commit -m "Message 1"
Staging Area에 있는 파일을 Repository로 이동
